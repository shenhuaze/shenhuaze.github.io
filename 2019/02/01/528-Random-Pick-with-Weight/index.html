528. Random Pick with Weight

Posted on 2019-02-01 | In Coding | | Visitors:

题目位置

https://leetcode.com/problems/random-pick-with-weight/

题目描述

Given an array w of positive integers, where w[i] describes the weight of index i, write a function pickIndex which randomly picks an index in proportion to its weight.

Note:

  1. 1 <= w.length <= 10000
  2. 1 <= w[i] <= 10^5
  3. pickIndex will be called at most 10000 times.

题目分析

给定一个整数数组w，每个元素代表权重，要求根据index的权重来随机选取index，换句话说，在随机选取index的时候，每个index被选中的概率正比于该index的权重。

参考http://www.cnblogs.com/grandyang/p/9784690.html的思想，构建一个前缀和数组。例如，假设w = [1, 3, 2]，那么相应的前缀和数组就是prefixSumW = [1, 4, 6]。然后随机选取一个0~5之间的随机整数x，找到prefixSumW数组中第一个大于x的元素，其index就是我们需要的index。
