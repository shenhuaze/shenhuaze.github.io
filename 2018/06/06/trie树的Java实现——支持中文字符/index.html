trie树的Java实现——支持中文字符

Posted on 2018-06-06 | In 自然语言处理 , trie | | Visitors:

之前实现了只包含英文字符的trie数，由于英文字母只有26个(就算考虑大小写，总共也只有52个)，因此可以用数组来实现，实现的trie树形式上看起来就是一棵26叉树。但由于中文字符有成千上万个，如果用数组来实现，会很浪费空间，因此这里采用HashMap的方式来实现trie树。

原理

这里实现trie树主要用到两个类：Trie和TrieNode。

TrieNode类主要用来表示trie树的每个节点，它有三个属性：当前节点的字符character、当前节点对应的子节点集合nodeMap、以及表示当前节点是否对应一个完整的词的boolean型变量state。此外，还有一个添加字符的方法addCharacter()。这里的子节点集合用HashMap来实现，key是某个子节点对应的字符，value是该子节点对应的TrieNode。

Trie类用来表示一棵trie树，它有一个TrieNode类型的属性rootNode，表示一棵trie树的根节点，根节点包含的字符是一个空字符。另外，还有一个添加单词的方法addWord()和匹配字符串的方法match()。匹配的返回状态有三种，未匹配(UNMATCH)、完全匹配(MATCH)和前缀匹配(PREFIX)，分别用整数0、1、2表示三种状态。
