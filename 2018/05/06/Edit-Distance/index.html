Edit Distance

Posted on 2018-05-06 | In Coding , Dynamic Programming | | Visitors:

题目位置

http://www.lintcode.com/en/problem/edit-distance/

题目描述

Given two words word1 and word2, find the minimum number of steps required to convert word1 to word2. (each operation is counted as 1 step.)

You have the following 3 operations permitted on a word:

  • Insert a character
  • Delete a character
  • Replace a character

Example

Given word1 = "mart" and word2 = "karma", return 3.

题目分析

  • 这道题求的是最少的步骤数，相当于求最小值。
  • 状态：dp[i][j]表示从word1的第i个字符转换到word2的第j个字符所需要的步骤数。
  • 方程：两种情况，一是当word1[i] == word2[j]时，dp[i][j] = dp[i-1][j-1]；二是当word1[i] != word2[j]时，dp[i][j] = min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1])。
  • 初始化：dp数组的维度为(word1.length+1, word2.length+1)，因为还要考虑空字符串的情况，初始化的时候就初始化第一行和第一列，dp[i][0] = i, dp[0][i] = j，为啥，因为空字符串转换到任意一个字符串的第i个位置时，肯定需要i步。
  • 答案：就是dp[word1.length][word2.length]。
