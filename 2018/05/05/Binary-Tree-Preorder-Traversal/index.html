Binary Tree Preorder Traversal

Posted on 2018-05-05 | In Coding , Divide Conquer | | Visitors:

题目位置

http://www.lintcode.com/en/problem/binary-tree-preorder-traversal/

题目描述

Given a binary tree, return the preorder traversal of its nodes’ values.

Example

Given:

return [1,2,4,5,3].

Challenge

Can you do it without recursion?

题目分析

给定一棵二叉树，返回它的前序遍历（根-左-右）。

注意

  • add到list中的是root.val，而不是root。
  • 非递归法。首先创建一个Stack和List。先把root节点push到Stack中，while Stack不为空的时候，先将Stack的最上面的节点pop出来，取出它的val。然后看pop出来的节点的右子节点，如果不为空，则将它push到Stack中；然后再看pop出来的节点的左子节点，如果不为空，则将它push到Stack中。
